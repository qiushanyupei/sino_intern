UE：User Equipment;Uu:和UE相关的无线接口，第一个U表示UE，第二个u通常表示无线接口<br>
eNB：eNodeB (Evolved Node B)，Node B本身是3G基站<br>
MME：Mobility Management Entity（移动性管理实体（控制面核心节点））<br>
S1-MME：eNB与MME的控制面接口，S1 中的 "S" 代表 SAE (System Architecture Evolution，系统架构演进)，"1" 代表这是该架构中定义的第一个、也是最重要的接口
SGW（Serving Gateway）：服务网关（用户面）
S1-U（S1 User Plane Interface）：eNB 与 SGW 的用户面接口
S11（S11 Interface）：MME 与 SGW 之间控制信令
HSS（Home Subscriber Server）：用户归属服务器（存储用户信息）
S6a（S6a Interface）：MME 与 HSS（鉴权/用户数据）
PGW（Packet Data Network Gateway）：分组数据网关（连接互联网）
PCRF（Policy and Charging Rules Function）：政策与计费规则功能
Gx（Gx Interface）：PCRF 与 PGW（策略控制）
SGi（SGi Interface）：PGW 到外部互联网
