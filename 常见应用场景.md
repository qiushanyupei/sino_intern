UE：User Equipment;Uu:和UE相关的无线接口，第一个U表示UE，第二个u通常表示无线接口<br>
eNB：eNodeB (Evolved Node B)，Node B本身是3G基站<br>
MME：Mobility Management Entity（移动性管理实体（控制面核心节点））<br>
S1-MME：eNB与MME的控制面接口，S1 中的 "S" 代表 SAE (System Architecture Evolution，系统架构演进)，"1" 代表这是该架构中定义的第一个、也是最重要的接口<br>
SGW（Serving Gateway）：服务网关（用户面）<br>
S1-U（S1 User Plane Interface）：eNB 与 SGW 的用户面接口<br>
S11（S11 Interface）：MME 与 SGW 之间控制信令<br>
HSS（Home Subscriber Server）：用户归属服务器（存储用户信息）<br>
S6a（S6a Interface）：MME 与 HSS（鉴权/用户数据）<br>
PGW（Packet Data Network Gateway）：分组数据网关（连接互联网）<br>
PCRF（Policy and Charging Rules Function）：政策与计费规则功能<br>
Gx（Gx Interface）：PCRF 与 PGW（策略控制）<br>
SGi（SGi Interface）：PGW 到外部互联网<br>
然后是下面的产品内部分的system，依靠NP卡来区分<br>
前情提要：LTE网络4G 无线接入网（就是4G公网）（RAN）（Radio Access Network）。Long-Term Evolution（长期演进技术）。主要应用于<br>
系统1：传统 LTE 宏网络系统（宏网络是指大范围、广域连续覆盖的移动通信网络），作为主路径（anchor），负责控制面（如通过 MME）和部分用户面。<br>
系统2：第二接入系统（如 WLAN / 非3GPP接入）；接口和system1类似，承载分流路径（offload path），与 System 1 做上行/下行数据聚合<br>
手机通过蜂窝网（4G/5G）连上运营商网络 → 这就是 3GPP 接入
非 3GPP 接入：WiFi、有线宽带、WiMAX 等
系统3：VoLTE / IMS 系统；通常不分流，保持高 QoS 路径；但汇聚场景下可能指示保证语音业务仍走主路径，数据业务走分流。<br>
系统4：业务/互联网系统（IP Services）；分流汇聚的终点——聚合后的流量最终去往 IP 网络<br>
